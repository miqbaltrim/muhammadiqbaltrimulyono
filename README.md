# Portfolio Multipage untuk GitHub Pages

## Isi folder
- `index.html` = halaman utama
- `about.html` = halaman tentang saya
- `projects.html` = halaman project
- `contact.html` = halaman kontak
- `assets/css/style.css` = stylesheet utama
- `assets/js/script.js` = animasi scroll reveal dan form demo
- `assets/img/profile-card.svg` = ilustrasi placeholder hero

## Cara deploy ke GitHub Pages
1. Buat repository baru di GitHub
2. Upload semua isi folder ini ke repository
3. Pastikan file utama bernama `index.html`
4. Buka `Settings > Pages`
5. Pada `Build and deployment`, pilih `Deploy from a branch`
6. Pilih branch `main` dan folder `/ (root)`
7. Save
8. Tunggu sampai link website muncul

## Yang perlu diganti
- `Nama Kamu`
- `emailkamu@example.com`
- `github.com/username`
- isi project, deskripsi, dan pengalaman
- huruf logo `P` pada brand mark jika diperlukan

## Catatan
GitHub Pages hanya cocok untuk website statis. Jika ingin form kontak benar-benar mengirim data, hubungkan ke layanan pihak ketiga seperti Formspree.
